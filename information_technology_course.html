Information Technology Programme Course

Thinking of becoming an IT specialist? This can be a very exciting opportunity for an individual who is passionate about Information Technology. You will get to learn how to use various technology tools, software and systems in order to solve problems related to information management.

MS Office

Microsoft Office is a suite of productivity applications for PCs. It includes word processing, presentation creation and editing, spreadsheet capabilities, and database management. In the Documents section, you can create and view documents that contain text or images. You can work with databases in the Database application to manage information such as client contact lists or inventory lists. The Data Connector section lets you connect to other computer applications and share data between them

Advanced Excel

The ability to use advanced Excel skills like creating graphs and tables, doing calculations and automation, and producing large volumes of data. The knowledge that comes with the skill set gives you an advantage over others because they may not have the know-how or know how to get it done in a way that meets your needs. You’re able to make the right decisions on what needs to be done, when it should be done, and how it should be done. You’ll also be able to train others in your organization on how to use Excel more effectively.

MIS (Management Information Systems)

A management information system is the heart of an organization's operations. It gathers data from multiple online systems, analyzes the information, and reports data to aid in management decision-making. A management information system (MIS) is a computer-based tool that provides organizations with information about their operations. The MIS receives data from multiple online systems, analyzes it using a variety of algorithms and rules, and then presents the results in an accessible form to managers. The goal of these reports is to help managers make decisions that will improve their business processes.

VBA - Excel Macros

Power through your work with VBA. Excel Macros are the perfect way to automate repetitive tasks and save time while they're at it. Save time while keeping your focus on the task at hand by creating automated processes in a matter of minutes that help you better organize your workflow. The most frustrating thing about Excel is how it can take hours to do something that should only take minutes. One of the best ways to combat this issue is by using macros. Macros are a VBA tool that lets you create automated processes in a matter of minutes that help you better organize your workflow.

MS-Access

Microsoft Access is a database management system and software development tool from Microsoft that integrates powerful database tools with tools to aid the development and deployment of Web applications. The suite includes the full Office client applications, as well as an integrated database and software development environment (IDE). This allows users to create and manage database applications that run on the desktop, in a web browser or on mobile devices. Microsoft Access is used by both professional developers and end users to create powerful database applications. With Access, it is possible to create everything from simple tables and forms to complex data-driven applications.

Tableau

Tableau is a software used to build and analyze maps, dashboards, timelines and charts in collaboration with others around the world. Users can create visualizations that tell stories, show relationships between ideas, help people in decision making, increase productivity and get actionable insights quickly. It is a powerful tool that can be used by individuals and businesses to gain insights into their data. The software comes with pre-built maps, charts and other visualizations that can help users gain quick insights into their data.

Power BI

Power BI makes it easy to bring your data together so you can discover new insights, and make effective decisions. Power BI represents your business in a visually engaging way. In one place, you can view relationships among all the assets, people and processes that are relevant to your business, without having to dig into multiple sources of data.

Python Data Analysis

Python data analysis or data wrangling is a programmatic technique of transforming, cleaning and processing data. This process includes organizing the data into a structured format and transforming it into a more suitable format to analyze. Data Wrangling consists of four main stages: receiving, cleaning, purifying and preparing data sets.

SQL

Structured Query Language (SQL) is a standardized programming language that is used to manage relational databases and perform various operations on the data in the database. SQL statements allow you to interact with your database, including managing and updating data (inserting, storing, retrieving, altering), and so on. A SQL statement ends with a semicolon and optionally includes a row or two for displaying results.

C Programming

With its low-level access to system memory and its ability to manipulate strings, pointers, arrays and advanced data types, C provides an excellent base for building highly customized programs. C is a procedural language that provides a stable platform upon which programmers can build computer software. To create an executable that a computer can run, a C program needs to be run through a C compiler. C is an imperative language, which means that it provides commands (or procedures) to instruct the computer how to perform tasks. C also has many features of a high-level language, including variables, loops and functions.

C++

C++ is a general-purpose programming language that is especially suited to cross-platform development, as well as object-oriented programming. It was originally developed by Bjarne Stroustrup beginning in 1979 at Bell Laboratories. C++ has gone through several evolutions, with each release adding new features but losing others. Each version has been a contributor to evolving standards such as the ISO/IEC standardization process and the Open Group's Common Language Infrastructure standard. It supports multiple modular programming approaches and generic programming, particularly in its standard library. Additionally, C++ is readily expanded with third-party libraries for network coding, image processing, sound handling, etc.
